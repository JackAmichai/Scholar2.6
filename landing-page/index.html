See it in Action

Scholar 2.6 doesn't just search; it thinks with you. Here is the workflow.

1. Conversational Refinement

The AI asks clarifying questions (Scope, Timeframe, Focus) to ensure it understands exactly what you need before searching.

2. Interactive Graph

Visualizes papers as nodes. Larger nodes are highly cited. Connections show citation relationships.

3. Semantic Discovery

Uses SPECTER2 embeddings to find papers that are conceptually similar, not just keyword matches.

Under the Hood

  • Semantic Scholar API Real-time access to 200M+ academic papers.
  • React Force Graph Physics-based simulation for natural data visualization.
  • Shadow DOM Injection Ensures the extension never breaks the style of the website you are visiting.
